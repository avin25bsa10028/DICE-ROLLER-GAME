ABOUT THE PROJECT:
------------------
This is a dice rolling game with a graphical interface.
You can roll 1 to 6 dice at once and see the total score.
The dice have animation effects to make it look realistic.

FEATURES:
---------
- Roll multiple dice (1-6)
- Animated rolling effect
- Shows total score
- Easy to use interface
- Cool graphics

HOW TO RUN:
-----------
1. Make sure Python is installed on your computer
2. Install Pillow library:
   Open command prompt and type:
   pip install Pillow

3. Run the program:
   python dice_roller.py

LIBRARIES USED:
---------------
- tkinter (for making the window and buttons)
- PIL/Pillow (for creating dice images)
- random (for random dice numbers)

HOW TO USE:
-----------
1. Open the program
2. Select how many dice you want (1 to 6)
3. Click the "ROLL DICE" button
4. Watch the dice roll!
5. See your total score

SYSTEM REQUIREMENTS:

- 2GB RAM minimum
- Pillow library installed
